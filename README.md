# Diabetes Prediction Model
Predicts the likelihood of type 2 (non-childhood) diabetes based on family history of certain traits.

# Languages
Built using Python.
Dataset from Kaggle.

# Process

1. I ran Exploratory Data Analysis on the dataset, checking for shape, columns, null entries, and correlation between columns.
2. I implemented Multiple Linear Regression, since there were many traits that contributed to the outcome, and Polynomial Regression, since there may not be a linear pattern to the dataset. After implementing both, I concluded that Polynomial Regression would be a better regression for this dataset.
